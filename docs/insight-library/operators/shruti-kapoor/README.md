# Shruti Kapoor

## Bio
Shruti Kapoor. Founder coach helping early-stage founders move from mental clutter to meaningful action. Founded Wingman in Bangalore in 2018 (YC S19), a conversation-intelligence platform for sales teams that was acquired by Clari in 2022. Post-acquisition led international business for Clari across EMEA & APAC, then left to start The Momentum Studio. IIM-A alum. Public reference voice on weathering downturns from seed stage and on co-founder alignment.

## Operating themes
- **Listening mode beats selling mode in a downturn**, use contraction as structured user research.
- **Talk in customers' revenue, not your own**, reframe pitches around the buyer's funnel math.
- **Co-founder misalignment is the silent killer**, empower questioning across function lines.
