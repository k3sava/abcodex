# Claire Hughes Johnson

## Bio
Claire Hughes Johnson. Former Chief Operating Officer at Stripe, where she scaled the company from roughly 160 to 7,000+ employees. Previously held leadership roles at Google including the self-driving project (now Waymo) and Global Online Sales. Author of *Scaling People*, the canonical operator manual for installing operating systems into growing companies.

## Operating themes
- **Force for positive momentum.** When the decider is unclear, you are it. Type 2 decisions reward motion.
- **The company-as-house metaphor.** Foundation, supporting beams, mechanicals, install all three before scale forces it.
- **Make the implicit explicit.** Why, who decides, what criterion, who is informed.
- **Hypothesis-based coaching.** A few traces and a hypothesis beats waiting for statistical significance.
