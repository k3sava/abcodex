# Boris Cherny

## Bio
Boris Cherny. Head of Claude Code at Anthropic. Took the product from 0 to roughly $2B run-rate. Previously at Meta (Facebook). Public face for Anthropic's "the product is the model" design philosophy and the operator-grade use of Claude Code internally, claims 100% of his own code is written by Claude since November 2025, with ~5 agents running in parallel and 10–30 PRs a day.

## Operating themes
- **The product is the model.** Minimal scaffolding around the LLM. Tools + goal, not workflows.
- **Underfund deliberately.** One person plus the substrate beats four people without it.
- **Build for the model six months out.** Accept weak PMF at launch; inflect when the next model lands.
- **Latent demand is the search algorithm.** Watch what users hack around the product to get a job done.
