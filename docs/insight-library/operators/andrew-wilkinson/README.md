# Andrew Wilkinson

## Bio
Andrew Wilkinson. Bootstrapped operator. Founded or involved with 75 businesses across nearly every business model. Built MetaLab, then assembled Tiny, a holding-company-of-bootstrapped-cash-flow-businesses including Letterboxd, Dribbble, Serato, and others. Vocal counter-narrative to venture-backed startup orthodoxy: pick boring undermonetized niches, delegate ruthlessly, and stay private long enough to compound.

## Operating themes
- **Fish where the fish are.** Boring markets with willing buyers beat sexy markets with chewed-up incumbents.
- **Lazy leadership.** Delegate every task you hate; fight the Protestant work ethic.
- **Willingness to pay tracks buyer economics.** Same service, different segment, very different price.
