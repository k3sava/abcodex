# James Doman-Pipe

## Bio
James Doman-Pipe. Product marketing consultant who runs Building Momentum, a blog and newsletter helping B2B SaaS startups with positioning, messaging, and GTM. Created the Customer Value Positioning (CVP) framework, a 4-step process (Discover Buyer Persona → Describe Value → Detail Product Fit → Define Positioning) that produces actionable deliverables including homepage copy. Maintains a curated swipe file of 680+ product marketing articles. Emphasis on customer interviews as non-negotiable, selling to situations rather than job titles, and concrete outputs over abstract positioning docs. Closely aligned with VPC-driven methodology (VPC-based, evidence-driven positioning).

## Operating themes
- **Operating thesis:** Positioning must be customer-centric, not internal-assumption-driven, interviews are non-negotiable, and the output should be actionable homepage copy, not abstract documents
