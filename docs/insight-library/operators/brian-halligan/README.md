# Brian Halligan

## Bio
Brian Halligan. Co-founder and longtime CEO of HubSpot, now an in-house CEO coach at Sequoia Capital. Public reference voice for the LOCK+S CEO evaluation rubric, the spiky-vs-un-weakness hiring frame, and peer-room learning models for executives.

## Operating themes
- **LOCK + S rubric.** Lovable, Obsessed, Chip-on-shoulder, Knowledge, Student.
- **Hire spiky.** Real strengths and real weaknesses beat well-rounded averages.
- **Hire slow, fire fast.** Most teams do the opposite.
- **Peer rooms learn faster than coaches.** Misery loves company; same-stage CEOs teach each other best.
- **Distribution is the new moat.** Starting has never been easier; scaling has never been harder.
