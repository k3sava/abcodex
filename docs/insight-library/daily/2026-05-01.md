# Phase 1, codex foundation, 2026-05-01

The repository had been live since March 3, but May 1 is when it became a real corpus. The reframing came first: from a personal study log to "an open, primary-source library of operator insights."

## What landed

Counts (verified from `git log --diff-filter=A`): **109** insight cards, **174** operator profiles, 1 card modification.

- **Site foundation**, Phase 1 build. Core pages: home, operators, browse, timeline, about. INDEX.json + INDEX.md generation pipeline. Ingest protocol + insight-card schema documented. UI brief written.
- **Watch OS-inspired knowledge graph**, the force-directed map view shipped with this UI. Operators outside, domains in the middle, insights orbiting between.
- **GitHub Pages workflow**, automated deploy on push to `main`.
- **30+ ingest commits**, each named after the operator(s) being added. The first cards from a long list of voices that would shape the corpus going forward:
