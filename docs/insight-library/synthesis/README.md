## Method

- Pattern minimum: 3 distinct operators with cards that converge on the same substantive claim. Multiple cards from the same operator count as one operator-vote.
- Contradiction minimum: 2 operators with cards that genuinely disagree on a substantive question (not just framing differences).
- Source: 150 insight cards in `insights/` as of INDEX.json captured 2026-05-01.
- Built by Batch S1 (synthesise pass over the corpus).
