## Maturity levels
- Foundational — durable principle, holds across eras
- Applied — current best practice, eras-bound
- Frontier — emerging, under active testing

## Artifact classes
- Framework — model or mental scaffold
- Playbook — sequenced steps with conditions
- Template — fill-in-the-blank artifact
- Workflow — recurring operating loop
- Metric Model — what to measure and why
- Case Study — what an operator shipped, with outcome
- Transcript Excerpt — verbatim quote with timestamp + context
