# Podcast transcripts can be turned into reusable AI agent skills for practical work.

## Claim
Instead of building another chatbot, we extracted knowledge from Lenny's podcast transcripts and turned them into Agent Skills that people can use immediately in the places they work.

## Mechanism
By downloading transcripts and converting insights into structured skills, users can install them into Claude or any AI to get access to that expertise in their daily work, e.g., a Product Vision skill that generates documents or gives feedback.
