# A graveyard of incomplete agents is where the good stuff lives

## Claim
That graveyard of failed experiments, weird ideas, and rough drafts is where the breakthroughs live. If you're not failing, you're not building.

## Mechanism
The graveyard represents a backlog of learning and iteration. Each failure is a stepping stone to a successful agent, so the willingness to ship incomplete work accelerates discovery.
