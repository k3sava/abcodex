# Branching chats with stable taxonomy scales PMM strategy without chaos.

## Claim
Using 'Branch in new chat' with a consistent prefix taxonomy (STR, MSG, PER, etc.) lets product marketers spin off focused workstreams from a single context thread.

## Mechanism
Branches preserve context while isolating tasks, preventing cross-contamination. A stable taxonomy enables search, reuse, and custom GPT referencing. A 'Base Context' branch with pinned summaries ensures consistency across branches.
