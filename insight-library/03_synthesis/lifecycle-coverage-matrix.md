# Lifecycle Coverage Matrix

## Stages
1. Discovery (SEO/AEO, social, community)
2. Consideration (positioning, proof, enablement)
3. Conversion (CRO, objection sequencing, pricing clarity)
4. Onboarding (activation messaging, docs, support handoff)
5. Adoption (use-case expansion, role success plans)
6. Expansion (upsell/cross-sell narrative, business value proof)
7. Renewal/Advocacy (QBR narrative, champion enablement, community loops)

## Functional coverage
- Product: positioning feedback, onboarding instrumentation, feature adoption signals.
- Sales: discovery talk tracks, objection handling, stage progression assets.
- Customer Success: value realization frameworks, renewal narratives, expansion kits.
- Support: ticket intelligence loops, documentation optimization, deflection workflows.
- Customer outcomes: time-to-value, ROI proof, adoption depth, champion creation.

## Current status
- Strong: PMM/GTM narrative systems, AEO foundations, demand-gen loops.
- Emerging: CRO, CS, support lifecycle signal integration.
- Gaps remaining: lifecycle benchmarking standards, vertical-specific lifecycle playbooks.
