# AI-Native PMM/GTM Insight Library

This Markdown library captures thought-leader insights and converts them into reusable PMM/GTM operating assets.

## Current state
- Governance layer is defined (thesis, taxonomy, scoring, source-credibility protocol).
- Expert map expanded toward first-principles thinkers + AI-native system builders.
- High-volume ingestion layer added for **300+ sources** and **900+ atomic insight candidates**.

## Scaled ingestion system
- `06_ingestion/source-master-index.md` -> source queue and metadata.
- `06_ingestion/insight-backlog.md` -> atomic insight backlog for scoring/promotion.
- `06_ingestion/ingestion-sprint-plan.md` -> weekly throughput and quality gates.

## Operating loop
1. Catalog primary sources.
2. Extract atomic insights.
3. Score and tag insights.
4. Promote highest-value patterns into synthesis and playbooks.
5. Map promoted insights to Commons/Flywheel/clearPMM artifacts + signals.

## Definition of done for each promoted insight
- Clear claim
- Mechanism
- Application conditions
- Measurable signals
- Explicit OS mapping
