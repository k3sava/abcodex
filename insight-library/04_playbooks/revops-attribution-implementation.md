# RevOps Attribution Implementation

## Goal
Operationalize dual-layer attribution for execution speed and strategic accuracy.

## Layer 1: Weekly operating view
- Last-touch / self-reported attribution for rapid optimization.

## Layer 2: Monthly strategy view
- Cohort-based multi-touch influence model for budget and channel strategy.

## Implementation checklist
1. Define canonical event schema.
2. Map campaign/source IDs to assets in Commons.
3. Establish cohort definitions.
4. Build QA checks for missing/duplicate events.
5. Review conflicts and document decisions monthly.
