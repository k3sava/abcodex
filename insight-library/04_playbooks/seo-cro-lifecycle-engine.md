# SEO + CRO Lifecycle Engine

## Objective
Unify discovery-to-conversion optimization so organic/AI visibility connects to qualified pipeline.

## Workflow
1. Build intent clusters (problem-aware, solution-aware, comparison, decision).
2. Map each cluster to page intent, proof type, and CTA path.
3. Track visibility metrics (mentions/citations/rank) with conversion metrics (CVR, SQL rate).
4. Run objection-order tests on high-intent pages.
5. Feed learnings into sales and success messaging.

## KPIs
- AI mention share
- Organic qualified session rate
- LP conversion by intent
- Opportunity progression after content exposure
