# Using ChatGPT for passive-aggressive messages is better than for positive ones

## Claim
Using ChatGPT to write positive messages to your colleagues is a no-go, but using it to write passive-aggressive messages is perfect.

## Mechanism
The humor implies that AI is better suited for crafting subtle, indirect criticism than for genuine warmth, because the latter requires authentic human sentiment.
