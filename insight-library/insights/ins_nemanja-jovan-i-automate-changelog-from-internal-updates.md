# Automate changelog creation from internal PM updates using AI.

## Claim
We built an automation that turns internal Slack product updates into public changelog entries, reducing effort from 30 minutes to under a minute and ensuring consistent updates.

## Mechanism
By using n8n to trigger on Slack messages, AI checks if the update is public-facing and drafts a changelog entry in Canny following PMM guidelines. PMs still review before publishing, but the friction of manual drafting is removed, making it more likely to happen.
