# Persona-adapted content that actually differs requires parallel sub-agents and a similarity check.

## Claim
I built a Claude Code workflow with 5 sub-agents, each adapting the same source material for a different persona, then a coordinator agent rejects any version more than 40% similar in structure.

## Mechanism
Most persona-adapted content sounds the same because it's hard for one person to hold multiple perspectives. By running agents in parallel and enforcing a strict similarity threshold, each version gains distinct pain points, problems, and desires for its specific persona.
