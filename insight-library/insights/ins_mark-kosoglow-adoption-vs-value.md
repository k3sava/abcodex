# Tool adoption should not be conflated with value delivery.

## Claim
A GTM tool can deliver value even if zero reps log in, as long as it integrates into existing workflows.

## Mechanism
Value comes from data flowing into the systems where reps already work, not from forcing them to log into another platform. Measuring adoption by login frequency misses the real impact of invisible, embedded tools.
