# Vibe-coding a lightweight internal tool solves the 'I need a customer who' request gap.

## Claim
I built a simple internal tool that lets GTM teammates type in specific customer criteria (e.g., 'Midmarket & enterprise customers using Customer Hub I can name-drop') and returns matching customers with health score, ARR, products used, case studies, and more.

## Mechanism
The tool works by pulling structured data from a Tableau dashboard into a CSV, then using a simple interface to filter and surface relevant customers. This fills the gap between formal reference management tools and quick, early-stage customer discovery requests.
