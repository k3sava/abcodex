# Adriel Frederick

## Bio
Growth operator with leadership tenures at Lyft, Facebook, and Instagram. Public reference voice for marginal-user methodology, the human-in-the-loop framing of algorithmic systems (humans set constraints, algorithms amplify intent), and the lead-bullet / cannonball portfolio discipline.

## Operating themes
- **Marginal user.** Design for the person on the cusp of converting in the worst conditions.
- **Watch the user, don't just read the data.** Funnels show where; only watching shows why.
- **Lead bullets and cannonballs.** Mix many small bets with rare large ones; resist the lead-bullet laziness trap.
- **Diverse teams accelerate user research.** A team that resembles the user base debates faster than recruited panels.
