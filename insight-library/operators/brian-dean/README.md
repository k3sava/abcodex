# Brian Dean

## Bio
Brian Dean built Backlinko into one of the most authoritative SEO resources on the internet by practicing what he taught: creating content so comprehensively useful that it attracted links naturally while also being systematically promoted through outreach. His signature contribution is the Skyscraper Technique, a three-step link building methodology: (1) find content that already has significant backlinks using tools like Ahrefs, (2) create something substantially better along dimensions of depth, freshness, design, or data, and (3) reach out to sites that linked to the original piece and suggest they link to your superior version instead.

## Operating themes
- **Operating thesis:** SEO success comes from creating content that genuinely deserves to rank number one, then systematically promoting it to people who can amplify it; the best link building is a byproduct of being the best resource on a topic.
