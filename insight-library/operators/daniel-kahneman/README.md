# Daniel Kahneman

## Bio
Daniel Kahneman's foundational contribution is the two-system model of cognition that explains why intelligent people make predictable errors. System 1 operates automatically, rapidly, and effortlessly, generating impressions, feelings, and inclinations that feel like truth. System 2 is the deliberate, analytical, resource-intensive process that can override System 1 but rarely bothers to, because it is inherently lazy and defaults to endorsing whatever System 1 suggests.

## Operating themes
- **Operating thesis:** Human judgment is systematically flawed by predictable cognitive biases; the overconfident System 1 generates intuitions that System 2 is too lazy to override, and the result is that we are reliably wrong in identifiable patterns.
- **Cognitive Bias Management**
- **Decision Quality Framework**
