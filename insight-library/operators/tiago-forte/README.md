# Tiago Forte

## Bio
Tiago Forte's Building a Second Brain methodology provides the foundational architecture for how knowledge workers should capture, organize, and retrieve information -- a system that becomes dramatically more powerful when paired with AI. His central framework is CODE: Capture (save only what resonates, using a "resonance" filter rather than trying to capture everything), Organize (file by actionability using the PARA method, not by topic), Distill (progressively summarize notes through multiple layers of compression), and Express (turn accumulated knowledge into creative output and concrete results).

## Operating themes
- **Operating thesis:** Our brains are for having ideas, not storing them -- an external, organized knowledge system compounds over time and becomes the foundation for creative output, especially when AI can now search, synthesize, and surface that knowledge on demand.
