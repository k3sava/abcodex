# 3 Pillars to Ramp Up User Adoption and Unlock Customer Value

Archived Gartner research (Charlie Xu, John Quaglietta, Jennifer MacIntosh) on closing the user-adoption gap. Core finding: 60% of tech users say new applications were introduced without explanations of why and how they would help — buyers default to assuming purpose is self-evident, which kills adoption.

## Key takeaways

- Three pillars: build user anticipation, motivate value-critical actions, build proficiency through contextual + real-time training.
- "Why" change statements must precede capability rollout — assumed self-evidence is the default failure mode.
- Adoption is a retention and growth lever, not just an onboarding task.
- Real-time + contextual training (in-product) outperforms classroom/LMS for SaaS.

## Quotes worth keeping

> "Sixty percent of tech users report that new applications were introduced without explanations of why and how they would help them." (p.1)
