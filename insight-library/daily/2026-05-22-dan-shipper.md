Shipper published a full transcript of his AI and I podcast episode with Stainless CEO Alex Rattray, recorded after Anthropic's $300 million acquisition announcement. The conversation centers on MCP, the Model Context Protocol, and how it functions as the connective tissue between AI agents and the rest of the internet. Rattray explains that Stainless built SDKs that let developers generate type-safe client libraries from OpenAPI specs. With MCP, that same approach extends to letting AI systems call any API as a tool, not just a data source. The acquisition gives Anthropic direct ownership of the protocol layer underneath third-party integrations. For substrate, this is directly relevant. MCP is no longer a community standard that might or might not get maintained. It is now a first-party Anthropic investment. Building substrate's agent integration layer on MCP is now a lower-risk bet than it was two weeks ago. For ab-codex, MCP-based connections to data sources (operator blogs, feeds) become more plausible as a long-term ingestion architecture.
