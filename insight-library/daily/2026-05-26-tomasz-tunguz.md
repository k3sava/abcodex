Tomasz Tunguz's May 22 post observes that Salesforce has gone headless: a sales rep can update a deal without opening salesforce.com because AI routes the action through natural language. His framing: AI makes the user interface plastic, malleable to what the user needs, when they need it. The head does not disappear; it becomes formless. Software companies that treat their UI as their moat are now exposed. Those whose moat is data and API surface are fine.

For ab-codex and consulting: ab-codex.iamkesava.com does not have a locked UI moat, which is an advantage. The insight library's value is in the data and operator attribution layer, not the page wrapper. This also sharpens the case against building heavy front-end differentiation for any of Kesava's tools. Invest in data quality and API surface; the interface can be composed around it.
