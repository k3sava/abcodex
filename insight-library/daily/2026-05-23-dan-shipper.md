Shipper covered Anthropic's "Code with Claude" developer conference in SF and led with an observation worth keeping: the biggest announcement wasn't a model and wasn't a feature. It was a compute deal. Anthropic rented all 220,000 GPUs of SpaceX's Colossus 1 cluster for $1.25 billion per month. The direct result for users: Claude Code limits doubled for Pro, Max, Team, and Enterprise plans. Netflix was named as a multiagent production user. When the headline story from an AI developer conference is a real estate and compute infrastructure deal, the capability debate has moved on. The bottleneck now is capacity, not what the model can do.

For ab-codex: Shipper's editorial frame is worth learning from. He didn't recap the feature list. He identified what the announcement actually signaled about Anthropic's constraint. That's the level ab-codex should operate at. For substrate: doubled Claude Code limits matter immediately for build velocity. For consulting: Netflix as a named production multiagent customer is a reference case worth having in the pitch deck.
