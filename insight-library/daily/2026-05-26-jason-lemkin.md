SaaStr's AI Agent API Report Card now covers 144 B2B APIs, graded on six dimensions: authentication, error handling, documentation, pagination, idempotency, and agent-friendliness. The average score is 71/100. Stripe is A+. Most APIs pass on two or three dimensions and fail silently on the rest. The point Lemkin makes is that most B2B software was designed for humans to call, not agents, and the gap shows immediately when you try to integrate autonomously.

For substrate and consulting: this is a practical evaluation tool for any client or project choosing B2B APIs to build agentic workflows on top of. Before recommending a vendor stack, score their API against these six dimensions. For ab-codex specifically: if any insight library ingestion pipelines depend on third-party APIs, agent-friendliness of those APIs will determine how reliable the automation becomes at scale.
