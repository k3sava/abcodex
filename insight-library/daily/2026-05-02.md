# Pass C orphan ingestion + Curiosity backfill — 2026-05-02

## Curiosity backfill — external links

Curiosity (the personal KB) had operator URLs that hadn't been propagated into ab-codex profiles. Eight backfill batches ran across the day, adding external links (LinkedIn, blog, newsletter, website) for roughly 200 operator profiles. Each batch was 22-25 operators.

## Launch-ready scrub

A separate audit pass closed the gaps that orphan ingestion was likely to introduce:
- 8 stubs removed (cards that hadn't reached substance bar).
- 30 URLs validated (some had drifted; replacements found where possible).
- 73 cards repaired (frontmatter fixes, missing fields filled, attribution corrections).

Plus targeted scrubs: removing personal annotations from raw source files, pruning personal-network operators, scrubbing private raw frontmatter, splitting joint Hamel/Shreya attribution, removing legacy-batch directory.

A messy, productive day that turned the corpus from "early" to "launch-ready."
