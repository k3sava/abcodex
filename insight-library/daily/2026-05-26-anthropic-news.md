Code with Claude London ran May 19, oversubscribed, at a riverside venue. Anthropic shipped five features instead of a new model: Dreaming (Claude Code agents write notes to themselves so subsequent agents inherit context faster), Outcomes (goal-level task tracking), multi-agent coordination, Claude Finance (10 pre-built agents), and Add-ins. Fortune reported half the packed room raised their hands when asked if Claude had written their last PR entirely. MIT Technology Review framed the event as Anthropic showing the future of coding "whether you like it or not."

For substrate and career-ops: Dreaming is the feature to watch. It is Anthropic's answer to context loss across long-running agentic work. If substrate builds any multi-step Claude Code workflows, Dreaming is the architecture assumption to design around now. Claude Finance signals Anthropic moving into vertical agent bundles, which means the substrate differentiation needs to sit at the operator-data layer, not model access.
