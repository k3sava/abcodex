On May 19, KPMG and Anthropic announced a global alliance embedding Claude into KPMG's Digital Gateway platform, making Claude available to all 276,000-plus KPMG employees worldwide. The initial focus is tax clients and private equity firms. A new offering called KPMG Blaze embeds Claude Code to help clients cut IT modernization timelines and accelerate software delivery. The same week, May 18, Anthropic acquired Stainless for approximately $300 million. Stainless built SDK tooling and MCP server infrastructure, the protocol layer that lets Claude agents connect to external systems. Two moves in four days: one captures the enterprise deployment channel at Big Four scale, the other secures the developer infrastructure layer underneath it. For consulting, the KPMG deal signals that Claude proficiency will soon be a baseline expectation in professional services engagements, not a differentiator. For ab-codex and substrate, the Stainless acquisition means Anthropic is now investing in the MCP ecosystem directly, which makes MCP a safer long-term bet for agent integrations than it was last month.
