# Thought Leader Profiles

Each leader folder must contain:
- `profile.md`
- `sources.md`
- `insights.md`
- `frameworks.md`
- `contradictions.md`

## File responsibilities
- `profile.md`: why this voice matters + areas of expertise.
- `sources.md`: canonical source list (primary first, dated).
- `insights.md`: atomic insight cards only.
- `frameworks.md`: distilled reusable models/checklists.
- `contradictions.md`: where this leader conflicts with others.

- Leader:
