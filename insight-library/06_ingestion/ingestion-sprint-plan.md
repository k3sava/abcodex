# Ingestion Sprint Plan for 100s of Sources

## Throughput target
- 300 sources indexed
- 900+ atomic insight candidates logged
- 200 scored insights/week
- 40 promoted insights/week into leader + domain files

## 4-step weekly cadence
1. **Collection (Mon):** add 60-80 new source candidates.
2. **Extraction (Tue-Wed):** convert each source into 3-7 atomic insight candidates.
3. **Scoring (Thu):** rank candidates by originality, specificity, evidence, transferability, OS alignment.
4. **Promotion (Fri):** move top insights into thought-leader and playbook files with metrics and mappings.

## Quality gates
- No insight promoted without measurable signal definition.
- No source promoted without at least one primary link.
- Every promoted insight must map to Commons + Flywheel + clearPMM.

## Batch tags
- batch-001 to batch-010 (first 10 weeks)
- owner, date, confidence, and domain required per item
