# Source and Insight Scoring Model

## Scoring dimensions (1-5 each)
1. Originality: introduces non-obvious thinking.
2. Operational specificity: includes steps or implementation detail.
3. Evidence depth: contains proof points, examples, or data.
4. Transferability: can be adapted beyond a single context.
5. OS alignment: maps cleanly to Commons/Flywheel/clearPMM.

## Quality bands
- 22-25: Tier A (core canon)
- 17-21: Tier B (strong, selectively reusable)
- 12-16: Tier C (situational)
- <12: Archive only
