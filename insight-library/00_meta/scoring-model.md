# Insight Scoring Model

## Scoring dimensions (1-5 each)
1. **Originality** — introduces non-obvious thinking; not a restatement of common wisdom.
2. **Operational specificity** — includes steps, conditions, or implementation detail.
3. **Evidence depth** — proof points, examples, data, or first-hand experience.
4. **Transferability** — adaptable beyond the operator's specific context.
5. **Source integrity** — primary, attributed, dated, link-verifiable.

## Quality bands
- 22-25: **Tier A** — core canon, surface in any relevant query
- 17-21: **Tier B** — strong, selectively reusable
- 12-16: **Tier C** — situational, domain-specific
- <12: archive only

## Promotion gates
- An insight enters the corpus at Tier C or higher. Below that, archive in `raw/` only.
- Tier A requires at least two independent operator sources OR one primary source + a documented case outcome.
