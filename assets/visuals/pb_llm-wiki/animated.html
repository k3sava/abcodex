Annotated framework · four operations

The LLM Wiki

INGEST

Source → raw file → wiki stub → log entry

Source lands in raw/{subdir}/{slug}.md with frontmatter. Stub a synthesis page if warranted. Append one line to log.md. raw/ is append-only: never edit after creation.

QUERY

Index → 1–3 pages → cite, don't dump

Read index.md first. Read 1–3 relevant pages. Extract and cite paths. If the answer isn't there, say so. Do not fabricate. Re-reading pages you already read this session wastes tokens.

LINT

Walk every page. Fix high-impact in place.

Check: orphans, contradictions, stale claims, missing pages, naming drift, schema violations, index mismatches. Fix high-impact inline. Low-impact: document in lint/lint-NNN.md.

INDEX + LOG

One line per page. Append-only timeline.

index.md = content catalog, one line per page, under 200 lines. log.md = chronological, append-only, greppable. Wrong entry? Append a correction — never rewrite history.

Wire into sessions: @/path/to/wiki/system-prompt.md + @/path/to/wiki/index.md in CLAUDE.md →
